# Backend — Google Apps Script

This folder contains a reference copy of `Code.gs`.

The live backend runs at:
`https://script.google.com/macros/s/AKfycbxQ7HIztpGnAWRmhbTRvi-jFBOIChfZshH7fkTaBg81LDaDLfgmfXTOw5kqRJMWqMQ/exec`

**Do not deploy from this repo.** Manage the live script in the Google Apps Script IDE.

## Required Google Resources

- **SHEET_ID**: Google Sheet with tabs: MR_Visits, MR_Employees, Products, Auth
- **DRIVE_FOLDER_ID**: Google Drive folder for visit photo uploads

These are hardcoded in `Code.gs` and belong to the project owner's Google account.
